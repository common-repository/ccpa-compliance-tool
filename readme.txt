== Description ==

If you make money with your web site and you don’t protect your users, you could face huge fines. Now there is a way to protect your business, without expensive audits or consultant fees!

Privacy Pixel installs in under 4 minutes on any Wordpress site.  Instantly, you’ll be protected from lawsuits and fines for data privacy.  One click install.  Easy integration.  Simple setup in minutes.

== Features ==
* Plugin shows data opt-out modal to site visitors from California and European Union
* User gets the opportunity to opt-out, in compliance with CCPA/GDPR.
* If someone opts out, the plugin automatically destroys third party trackers across the site

Read more and get your subscription [here](https://www.privacypixel.com/)
FYI – We are using the Privacy Pixel API in order to provide our service. See the terms [here](https://privacy-pixel.webflow.io/terms-of-service/)

== Installation ==
1. Install and activate the plugin from the page WordPress -> Plugins.
2. Sign in to your account on the Private Pixel website and retrieve your API key.
3. Configure the plugin in the menu page on WP dashboard.
